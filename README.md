# Uber Eats Bangalore Restaurant Intelligence & Decision Support System

## Project Overview
This project analyzes Uber Eats Bangalore restaurant data using Python, MySQL, Pandas, and Streamlit. It provides an interactive dashboard for restaurant analysis and order analysis.

## Objective
- Analyze restaurant data
- Answer business questions using SQL
- Analyze order data
- Display results in an interactive dashboard

## 🛠️ Technologies Used
- Python
- Streamlit
- MySQL
- Pandas

## Project Files
- app.py
- mysql_connection.py
- business_questions.py
- order_business.py

## Features
- Dashboard
- Location Filter
- Cuisine Filter
- Rating Filter
- Business Questions
- Order Analysis

## How to Run

1. Install Python
2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
